# Quick Start: R Programming for Data Analysis

This repository is a collection of basic exercises and hands-on practices for learning the R programming language, specifically focused on data manipulation and statistical visualization.

## 📊 Core Learning Topics

### 1. R Basics & Environment
- **Setting Up**: Configured R and RStudio environment for statistical computing.
- **Basic Syntax**: Mastered variables, data types (Numeric, Character, Logical), and data structures (Vectors, Matrices, Data Frames).

### 2. Data Manipulation
- **Data Cleaning**: Practiced handling missing values and filtering datasets.
- **Data Structures**: Focused on using `Data Frames` to manage tabular data, similar to Excel spreadsheets.

### 3. Statistical Visualization
- **Base Graphics**: Created basic plots like histograms, scatter plots, and box plots.
- **ggplot2 (Optional/Planned)**: Introduction to the "Grammar of Graphics" for professional-grade visualizations.

## 🧪 Quick Practice Examples
- **Descriptive Statistics**: Calculating `mean()`, `median()`, and `sd()` for dataset analysis.
- **Visualizing Trends**: Generating simple linear regression models and plotting trend lines.

## 📂 Repository Contents
- `basics.R`: Script covering fundamental R syntax and operations.
- `visualization.R`: Code for generating various statistical charts.
- `/data`: Sample datasets used for analysis practice.

## 🛠 Tech Stack
- **Language**: R 4.x
- **IDE**: RStudio
- **Libraries**: base, datasets (tidyverse planned)
